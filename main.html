HTML tags

HTML Tags That We Have Learnt So Far



1. !DOCTYPE-This defines the document type

2. a-This defines a hyperlink

3. b-Defines bold text

4. br-This defines a single line break

5. button-Defines a clickable button

6. canvas-Used to draw graphics

7. center-Defines centered text

8. code-This defines a piece of computer code

9. div-Defines a section in a document

10. font-Defines font, colour and size for text

11. h1 to h6-Defines HTML headings

12. html-defines the root of an HTML document

13. img-Defines an image

14. input-Defines an input control

15. label-Defines a label for an input element

16. span-Defines a section in a document

17. var-Defines a variable

18. source-Defines multiple media recources for media elements

19. picture-Defines a container for multiple image recources
